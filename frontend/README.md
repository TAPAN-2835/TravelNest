# TravelNest — AI-Powered Travel Planning

Plan your dream trip in minutes with AI-powered itineraries, budget tracking, and destination discovery.

## 🚀 Features
- **AI Itinerary Generator**: Custom daily plans tailored to your interests and budget.
- **Discover**: Find trending destinations optimized for the Indian market.
- **Budget Tracker**: Track expenses in INR with real-time progress visualization.
- **Secure Vault**: Keep your travel documents safe and accessible.

## 🛠 Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express, Prisma (PostgreSQL), MongoDB, Redis.
- **AI**: Python (FastAPI), OpenAI GPT-4o-mini.

## 📦 Deployment
- **Frontend**: Ready for Vercel (includes `vercel.json`).
- **Backend**: Suitable for EC2/Heroku/Render.
